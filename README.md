# My First Android App with GitHub Actions

This project demonstrates automatic APK building using GitHub Actions.

## Features
- ✅ Automatic APK building on every push
- ✅ No local setup required
- ✅ Free CI/CD with GitHub Actions

## How to Build
1. Push code to this repository
2. GitHub Actions will automatically build the APK
3. Download APK from Actions tab

## Project Structure
- Standard Android project structure
- Ready for GitHub Actions
